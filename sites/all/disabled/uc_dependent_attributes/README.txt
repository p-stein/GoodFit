-- SUMMARY --

The Ubercart Dependent Attributes module allows store administrators to specify
which attribute/option combinations to enable/disable on a per product basis.

For a full description of the module, visit the project page:
  http://drupal.org/project/uc_dependent_attributes

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/uc_dependent_attributes


-- DEPENDENCIES --

* Ajax Attribute Calculations (http://drupal.org/project/uc_aac).


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Configure enabled/disabled attribute option combinations via the "Enabled"
  checkbox added to the product adjustments screen:
  /node/$nid/edit/adjustments
